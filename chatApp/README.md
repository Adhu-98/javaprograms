# Java Chat Application (Socket Programming)

## Overview
A simple client-server chat application using Java Sockets.

## Features
- Server handles multiple clients
- Client-server communication
- Real-time messaging

## How to Run

Step 1: Compile
javac ChatServer.java
javac ChatClient.java
javac ClientHandler.java

Step 2: Run Server
java ChatServer

Step 3: Run Client
java ChatClient

## Learning Outcome
- TCP communication
- Multi-threading
- Client-server architecture
