# Product Recommendation System

## Overview
A recommendation engine built using Java and Apache Mahout.

## Technologies
- Java
- Maven
- Apache Mahout

## How to Run
mvn clean compile
mvn exec:java -Dexec.mainClass=ProductRecommender

## Sample Output
Recommended products for User 1:
Product ID: 104 | Predicted Rating: 4.75

## Learning Outcome
- Collaborative filtering
- Recommendation algorithms
- Maven project structure
